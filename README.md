# xianyu_automatize
主要用于实现闲鱼真机自动化（包括自动签到、自动擦亮、统计宝贝数据）

一、文件含义：

	  1.appiumTest1.py是利用uu模拟器进行的闲鱼自动化操作，但由于模拟器上不能登录阿里系的应用，因此失败
	
	  2.xianyu.py包括自动签到、自动擦亮，连接真机运行
	
	  3.xianyuData.py则是用来统计宝贝数据
	
	  4.sort_list.p用来顺序储存每一个宝贝名
	
	  5.xianyuData.p用来存放当次统计的宝贝数据
